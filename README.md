# isingmodel

* **Prerequisites:**
	* Install SDL2:
		- "sudo apt-get install libsdl2-dev"
    
	* Innstall glfw3:
		- "sudo apt-get install libglfw3-dev"
		
To run:

gcc ising.c -lm

chmod +x DynSim.sh

./a.out | ./DynSim.sh -m 2 -l 100

#where the last number is the side of the lattice, which by default is 100
